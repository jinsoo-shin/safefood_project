# 안전 먹거리 (SafeFood) 프로젝트

## 개요
식품 별 첨가물과 영양성분 데이터를 분석하고, 식품 별 알레르기 유의사항을 표시
고객에게 안전한 먹거리를 선택 및 섭취를 도와주는 사이트

## 프로젝트 기간
2019.04.26 ~ 2019.05.16

## 개발 환경
- Java
- Spring
- MyBatis
- Mysql
- JSoup

## 담당 역할
풀스택 엔지니어 (기여도 50%)

## 담당 기능
- 식품 관련 API 제작
- 섭취 식품 API 제작
- 식품 검색 페이지 ㅂ
- 식품 정보 페이지 제작
- 게시판 페이지 제작
